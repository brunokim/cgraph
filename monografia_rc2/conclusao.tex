\section{Contribuições}

Este trabalho apresentou os resultados da simulação de disseminação de informação sobre redes complexas, 
apresentando os impactos que diferentes estruturas de redes implicam no sucesso de uma propagação. Para 
isto foi implementada e disponibilizada uma biblioteca de código aberto com capacidade para simulação de 
disseminação síncrona, além do tratamento e análise de vários \textit{datasets} disponíveis em diferentes 
bases de dados.

\section{Trabalhos Futuros}

Este trabalho apenas começou a explorar as possibilidades de propagação de informação. A biblioteca ainda
deve ser extendida para permitir propagações síncronas e com múltiplos contatos, para avaliar como a 
estrutura da rede altera tais resultados. Também devem ser avaliados os efeitos de diferentes pontos 
iniciais de propagação, e a seleção de tais pontos a partir de sua localização relativa na rede. Ainda,
deve-se avaliar o número total de interações, uma outra importante métrica de sucesso desconsiderada neste
trabalho.
