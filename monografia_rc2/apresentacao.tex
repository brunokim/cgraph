  \section{Modelos de rede}
  
  \begin{frame}
   \frametitle{Modelo de Erdös-Renyi}
   \begin{columns}
    \column{0.4\textwidth}
    \begin{figure}
     \includegraphics[width=\textwidth]{er.pdf}
    \end{figure}
    
    \column{0.6\textwidth}
    Parâmetros:
    \begin{itemize}
     \item $n$: número de vértices
     \item $k$: grau médio do grafo     
    \end{itemize}
    Para cada par de vértices, cria uma aresta com probabilidade $p = k/n$.
    \begin{itemize}
     \item Mundo pequeno: $d \sim \log n$;
     \item Transitividade é igual a $p$;
     \item Distribuição de graus: Poisson.
     \item Modelo muito simples, também chamado simplesmente ``aleatório''
    \end{itemize}
   \end{columns}

  \end{frame}

  \begin{frame}
   \frametitle{Modelo de Watts-Strogatz (1998)}
   \begin{columns}
    \column{0.4\textwidth}
    \begin{figure}
     \includegraphics[width=\textwidth]{ws.pdf}
    \end{figure}
    
    \column{0.6\textwidth}
    Parâmetros:
    \begin{itemize}
     \item $n$: número de vértices
     \item $k$: número de vizinhos aos quais cada vértice se conecta
     \item $\beta$: probabilidade de reconectar um extremo de cada aresta a outro vértice aleatório
    \end{itemize}
    Este modelo vai de um grafo regular ($\beta = 0$) a um aleatório ($\beta = 1$).
    \begin{itemize}
     \item Mundo pequeno: $d \sim \log n$;
     \item Transitividade entre $\frac{3}{4} \frac{k-2}{k-1} \leq C \leq \frac{k}{n}$;
     \item Distribuição de graus: Poisson;
     \item Modelo inclui fenômeno de mundo pequeno e alta transitividade.
    \end{itemize}
   \end{columns}

  \end{frame}
  
  \begin{frame}
   \frametitle{Modelo de Barabási-Albert}
   \begin{columns}
    \column{0.4\textwidth}
    \begin{figure}
     \includegraphics[width=\textwidth]{ba.pdf}
    \end{figure}
    
    \column{0.6\textwidth}
    Parâmetros:
    \begin{itemize}
     \item $n$: número de vértices
     \item $k$: número de arestas que cada novo vértice adiciona à rede.
    \end{itemize}
    Também chamado de \textit{rich-get-richer}, cada novo vértice se conecta, preferencialmente, aos vértices mais conectados.
    \begin{itemize}
     \item Mundo ultra-pequeno: $d \sim \log n / \log \log n$;
     \item Transitividade depende do tamanho da rede $C \sim n^{-3/4}$;
     \item Distribuição de graus: Pareto;
     \item Mecanismo de \textit{ligação preferencial} encontra paralelos em múltiplos sistemas.
    \end{itemize}
   \end{columns}
  \end{frame}
  
  \begin{frame}
   \frametitle{Modelo de Ravasz-Barabási}
   \begin{columns}
    \column{0.4\textwidth}
    \begin{figure}
     \includegraphics[width=\textwidth]{t3.pdf}
    \end{figure}
    
    \column{0.6\textwidth}
    Parâmetros:
    \begin{itemize}
     \item $l$: número de níveis de recursão
     \item $k$: número de vértices por subgrafo completo.
     \item $n$: $k^{l+1}$
    \end{itemize}
    Nós periféricos do nível $L-1$ se conectam ao nó central do nível $L$.
    \begin{itemize}
     \item Mundo pequeno: $d \sim l \sim \log n$;
     \item Transitividade inversamente proporcional ao grau do vértice: $C \sim k^{-1}$;
     \item Distribuição de graus segue lei de potência;
     \item Primeira inclusão dos três principais fenômenos em um modelo.
    \end{itemize}
   \end{columns}
  \end{frame}
